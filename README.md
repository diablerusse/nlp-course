# Прототип агентной системы: Суммаризатор учебника (Textbook Summarizer)

Данный репозиторий содержит итоговую практическую работу по проектированию интеллектуальных мультиагентных систем. Проект представляет собой прототип LLM-пайплайна, построенный на архитектуре условной оркестрации с роутером (Conditional Router).

## 🛠 Технологический стек
- **Язык:** Python
- **Фреймворки:** `langchain-core`, `langgraph`, `langchain-openai`
- **LLM Провайдер:** OpenRouter API (используются модели `deepseek/deepseek-chat` / `gpt-4o-mini`)

## 🎯 Описание функционала
Система анализирует пользовательский запрос и автоматически перенаправляет его наиболее подходящему агенту-эксперту:
1. **Анализ и классификация (Router Node):** Определяет намерение пользователя (выделение терминов, конспект, генерация вопросов и т.д.).
2. **Узел Definitions (Expert 1):** Извлекает строгие технические факты и составляет словари терминов.
3. **Узел Summary (Expert 2):** Генерирует развернутые, структурированные конспекты и обучающие материалы.
4. **Узел QA (General Expert):** Дает полезные практичные ответы на общие вопросы по тексту.

## 🏗 Архитектура пайплайна
```mermaid
graph TD
    A[Ввод пользователя] --> B[Маршрутизатор: Классификатор запроса]
    B -->|'definitions'| C[Узел Definitions: Извлечь только словарь]
    B -->|'summary'| D[Узел Summary: Написать общий конспект]
    B -->|'qa'| E[Узел QA: Ответить на точечный вопрос]
    C --> F[Сборка ответа и вывод]
    D --> F
    E --> F
    B -.->|Классификация LLM| LLM[LLM API]
    C -.->|Вызов LLM| LLM
    D -.->|Вызов LLM| LLM
    E -.->|Вызов LLM| LLM
